# etl-ia-generativa-python
Projeto demonstrando um pipeline ETL (Extração, Transformação e Carregamento) em Python, integrado com IA generativa para personalização de mensagens. Inclui boas práticas de engenharia de dados, testes automatizados e integração contínua (CI) com GitHub Actions — ideal para portfólio profissional em Ciência de Dados e Engenharia de Dados.
